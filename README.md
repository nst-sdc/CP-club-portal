## Requirements
- BeautifulSoup4

## Quick Start

1. Clone the repository:
    ```bash
    git clone https://github.com/nst-sdc/CP-club-portal.git
    ```

2. Navigate to the project directory:
    ```bash
    cd CP-club-portal
    ```

3. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

4. Run the scraper:
    ```bash
    python web_scraper.py
    ```
